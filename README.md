# 🚌 Сервис поиска оптимального маршрута между остановками

## 📌 Описание проекта

Данный проект представляет собой backend-сервис для поиска **оптимального маршрута между остановками общественного транспорта** с учётом:

- времени в пути  
- пересадок между маршрутами  
- расстояния между остановками  

В основе алгоритма лежит **модифицированный алгоритм A\***, адаптированный под транспортную сеть.

Проект разработан как учебный / демонстрационный, с упором на:
- архитектуру backend-приложения
- алгоритмы графов
- асинхронную работу с БД
- чистое разделение слоёв приложения

---

## 🧠 Основные возможности

- Поиск кратчайшего маршрута между двумя остановками
- Учёт пересадок с настраиваемым штрафом
- Поддержка нескольких автобусных маршрутов
- Расчёт времени в пути
- Формирование маршрута в удобном для отображения виде
- Визуализация маршрута на карте (HTML)

---

## ⚙️ Используемые технологии

- **Python 3.11**
- **FastAPI** — REST API
- **SQLAlchemy (async)** — работа с БД
- **Alembic** — миграции
- **PostgreSQL**
- **Docker / Docker Compose**
- **Алгоритмы графов (A\*)**
